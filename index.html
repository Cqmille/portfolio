ETT - Développement logiciel

ETT est un constructeur de centrales de traitement d’air à récupération d’énergie et de pompes à chaleur hautes performances. Afin de conserver ses certifications ou de pouvoir proposer à ses clients des solutions sur mesure, l'entreprise doit de posséder une solution de simulation performante qui rentre en adéquation avec le fonctionnement réel des machines.

Qu'il s'agisse de nouvelles machines à concevoir ou de nouveaux composants à intégrer, il est nécessaire de continuellement agrémenter le logiciel avec des DLL constructeur. C'est sur cette partie que j'interviens essentiellement.

Compétences clés
  • Utilisation des langages C# et VB sur le framework logiciel Microsoft .NET
  • Développement d'api ASP.NET permettant de faire l'interface entre la solution logicielle et les DLL constructeur
  • Création d'outils d'extraction de données, notamment pour la génération de polynômes avec Excel et Visual Basic
  • Fonctionnement projet en mode agile, organisation sur Azure Devops

Eureden - Développement Full Stack

Eureden est une société implantée à Landerneau et spécialisée dans l'industrie agroalimentaire.

Dans le cadre d'un stage réalisé dans l'entreprise, j'ai travaillé sur un système embarqué de gestion de commandes dont l'objectif est d'avertir l'utilisateur en cas de nouvelle entrée par des signaux sonores. Le système est également pourvu d'un écran permettant de renseigner l'utilisateur sur les détails de la commande à préparer.

Compétences clés
  • Conception d'un système embarqué sur Raspberry Pi avec Python
  • Développement d'api avec Laravel et MongoDB
  • Déploiement sous Docker
  • Création d'une interface utilisateur simple avec Vue.JS permettant de tester le fonctionnement du système
